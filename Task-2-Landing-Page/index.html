Everything between a request and a response.

// alerts

Alerts that name the cause

When p99 crosses your threshold, Wavelength attaches the deploy, the query, and the trace that triggered it, so the on-call engineer opens one link, not five dashboards.

// tracing

Follow a single request across services, queues, and databases. Every span is searchable by route, status code, or customer ID.
